Gestion de Bibliothèque - API REST Spring Boot

   Description
Ce projet est une **API RESTful** développée avec Spring Boot pour la gestion d’un système de bibliothèque.

Il permet d’effectuer des opérations CRUD complètes sur les entités suivantes :

- Auteurs
- Livres
- Catégories
- Éditeurs

Chaque livre est associé à un auteur, une catégorie et un éditeur à travers des relations entre entités.
## 📬 Collection Postman

Une collection Postman prête à l’emploi est incluse dans ce projet afin de tester facilement toutes les opérations CRUD.

   Nom du fichier :
CRUD-POSTMAN.json

   Comment l’utiliser :
1. Ouvrir Postman
2. Cliquer sur **Import**
3. Sélectionner le fichier : `CRUD-POSTMAN.json`
4. Exécuter et tester toutes les API (Auteur, Livre, Catégorie, Éditeur)

   Lien GitHub :
https://github.com/EddamiriAsmaa/GestionBibliotheque/blob/main/CRUD-POSTMAN.json
